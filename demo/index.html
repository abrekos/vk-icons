VK icons font

Иконки ВК в виде шрифта.

Как подключить и использовать?

1. Скачать шрифты с файлом стилей отсюда.
2. Нужно подключить файл стилей vk-icons.css
3. Используем лигатуры. Пример: <span class="vk-icon">newsfeed</span>
К иконкам применимы стили для текста.
